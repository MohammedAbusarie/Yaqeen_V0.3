Place the YaqeenScan executable here.
========================================

1. Copy your YaqeenScan .exe file into this folder.

2. Name the file exactly:  YaqeenScan.exe
   (If your file has a different name, e.g. YaqeenScan_debug.exe, either:
    - rename it to YaqeenScan.exe,  OR
    - update the download link in index.html to match your filename:
      find the <a href="./downloads/YaqeenScan.exe" ...> and change YaqeenScan.exe to your filename.)

3. The website will serve this file so users can download it from the "QR Deployable Attendance Tool" feature page.

Note: If you use a build name like YaqeenScan_debug.exe, you can keep that name and update the single link in index.html (search for "downloads/YaqeenScan.exe") to "downloads/YaqeenScan_debug.exe".
